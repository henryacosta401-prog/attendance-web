How To Test

1. Add a `Barcode ID` to an employee from the Employees page.
2. Print or display that barcode using `Code 128` if possible.
3. Choose the action here, then scan it from the phone camera.
4. This page uses the same attendance rules as the app, including split breaks and break limits.
