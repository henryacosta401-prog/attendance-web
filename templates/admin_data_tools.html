Recovery Checklist

1. Download the recovery pack before major cleanup, reset, or policy changes.
2. Create or verify the Render/Postgres provider backup outside the app, then mark it here.
3. Keep upload files and the Render disk safe together with database snapshots.
4. Remember that the workbook is a reference export, not an automatic restore button.
5. Review queued future schedule changes so rollout plans are backed up too.
